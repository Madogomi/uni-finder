# Uni-finder Web App
A simple React website for students to input their JAMB score and course and get all available universities.


## ⚙️ Tech Stack
- **React** – Core UI library
- **Vite** – Lightning-fast build tool
- **Tailwind CSS** – Utility-first responsive styling
- **React Router** – Client-side navigation
- **Animate On Scroll (AOS)** – Smooth scroll-based animations
- **Google Sheets** – Lightweight backend for data storage and synchronization
